# Lohit2104-S.K.-Lohit-Deveshwar-FactStream-Pdd

It is a daily affairs and interesting facts website developed for the younger adults where the user can upload contents about the current affairs and the interesting facts as well. Also, the user can upload only the real contents and on cannot upload the fake ones. For this, an API is integrated to check whether the uploading content is real or fake.
